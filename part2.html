1. function logUpTo(n) {
    for (let i = 1; i <= n; i++) {
    console.log(i);
    }
    }
    Time Complexity: O(n)
    The time complexity of the given function logUpTo(n) can be calculated as follows:
    In this function, we iterate from 1 to n and print each number. Therefore, the time complexity of the function is O(n), as it grows linearly with the input value n.
    So, the time complexity of this function is linear, increasing linearly with the input value n.
